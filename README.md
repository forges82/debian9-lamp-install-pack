# Debian 8 Lamp Install Pack

## First step

```
apt-get update
apt-get install git -y
mkdir /opt/script
cd /opt/script
git clone https://github.com/forges82/debian8-lamp-install-pack.git
cd /opt/script/debian8-lamp-install-pack
chmod +x install.sh
./install.sh
```
# Files

This file install Apache2, PHP 5 and MySQL.
